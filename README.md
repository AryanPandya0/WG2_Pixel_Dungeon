# Abyssal Depths - 2D Pixel Dungeon Crawler

A top-down 2D action-RPG built with HTML5 Canvas and Vanilla JavaScript.

## Features
- **Procedural Dungeon Generation**: Every run features a unique layout generated via random walker algorithms.
- **Pixel Art Visuals**: Custom-generated pixel art assets for characters, enemies, bosses, and environments.
- **3 Epic Bosses**: Face off against The Slime King, The Shadow Knight, and The Necromancer.
- **Custom Game Engine**: Smooth movement, particle effects, and screen shake.

## How to Play
1. Clone the repository.
2. Open `index.html` in any modern web browser or serve it using a local server (e.g., `python -m http.server`).
3. **Controls**:
   - **WASD / Arrows**: Move
   - **Mouse**: Aim
   - **Left Click**: Shoot

## Assets
The graphics in this game were generated using AI-driven pixel art techniques.
